# D&D Wiki — Guia de Classes para D&D 5e

**Guia de referência completo das classes de Dungeons & Dragons 5ª Edição em Português**

Guia em página única para as 14 classes de D&D 5e, totalmente em português. Cada classe possui tabela de progressão nível 1-20, descrição de habilidades, conjuração e subclasses.

**Classes**: Artífice, Bárbaro, Bardo, Bruxo, Caçador de Sangue, Clérigo, Druida, Feiticeiro, Guerreiro, Ladino, Mago, Monge, Paladino, Patrulheiro

### Funcionalidades

- Navegação com menu hambúrguer para mobile
- Tabelas completas de nível 1 a 20
- Subclasses listadas com sourcebooks oficiais
- Design responsivo (mobile, tablet, desktop)

### Como usar

```bash
git clone https://github.com/VGameleira/D-D-Wiki.git
cd D-D-Wiki
# Abra index.html no navegador
```
